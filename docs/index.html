Configura il Gestionale Cloud

Collega il gestionale a Firebase (gratuito, di Google) per usarlo su tutti i tuoi dispositivi con sincronizzazione automatica.

1
Crea un progetto Firebase gratuito

Vai su console.firebase.google.com, accedi con Google e clicca Aggiungi progetto. Chiamalo "gestionale-forno" e completa i passaggi.

2
Attiva Firestore Database

Nel menu laterale clicca Firestore Database → Crea database → Inizia in modalità test. Scegli la regione eur3 (Europe-west) e clicca Fatto.

3
Registra l'app web

Clicca su ⚙️ Impostazioni progetto → Le tue app → </>, inserisci un nome (es. "gestionale") e clicca Registra app. Vedrai un blocco con firebaseConfig = {'{...}'}.

4
Copia i valori di configurazione qui sotto

Incolla i valori dal blocco firebaseConfig nei campi seguenti.

⚠️ Compila tutti i campi obbligatori (API Key e Project ID).
Regola Firestore consigliata (Firestore → Regole → Modifica → Pubblica):
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /ordini/{doc} { allow read, write: if true; } } }
